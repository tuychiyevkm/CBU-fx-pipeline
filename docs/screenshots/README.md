# Screenshots

This folder holds the dashboard screenshots referenced by the project README.
They cannot be generated automatically — capture them in Power BI Desktop (or
the Power BI Service) **after the first successful data load**, then drop the
PNGs here using the exact filenames below.

| File              | Page        | What to capture |
|-------------------|-------------|-----------------|
| `overview.png`    | Overview    | The five KPI cards (USD, EUR, RUB, GBP, CNY) with green/red change indicators, the USD hero line chart, and the "Top movers today" table. |
| `history.png`     | History     | The currency slicer + date-range slicer, the line chart for a selected currency, and the detail table (date, rate_per_unit, pct_change). |
| `comparison.png`  | Comparison  | Several currencies normalized to 100 on one chart (with the reference line at 100), plus the sortable % change table. |

Recommended: 16:9, at least 1600×900, PNG, light theme (matches the brand
background `#F7F9FC`).

Once added, the README image links will resolve automatically.
